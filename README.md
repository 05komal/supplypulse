# SupplyPulse 

AI-powered multi-agent supply chain intelligence system.

## Features
- Multi-agent analysis (Disruption, Supplier, Demand, Finance)
- Risk scoring system
- Actionable recommendations
- Interactive dashboard

## Tech Stack
- FastAPI (Backend)
- HTML/CSS/JS (Frontend)
- Multi-agent architecture

## How to Run
uvicorn api.main:app --reload
